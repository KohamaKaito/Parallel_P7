# はじめに

このリポジトリは，並列処理によるソートプログラムの速度向上効果を実験するため，  
チームメンバー５人のソースコードの共有，バージョン管理を目的として作られました．


# 扱うアルゴリズム

・バケットソート  
・バブルソート  
・マージソート  
・クイックソート  
・シェルソート
